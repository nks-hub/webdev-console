## Security Findings Fixed in Iteration 4

Found and fixed through deep code review (not on todolist):

1. **Timing attack on auth** — `CryptographicOperations.FixedTimeEquals` with length pre-check
2. **Fragile Bearer parsing** — `StartsWith` + `Substring(7).Trim()`
3. **Sequential SSE broadcast** — `Task.WhenAll` parallelization
4. **Blocking File.ReadAllText in async endpoint** — `ReadAllTextAsync`
5. **Path traversal in `SiteManager.Delete/Update`** — `ValidateDomain` + `Path.GetFullPath` containment check
6. **Uncaught exceptions on DELETE/POST/PUT** — catch `ArgumentException`/`UnauthorizedAccessException` → HTTP 400
7. **SQL injection in `/api/databases/{name}/*`** — `IsValidDatabaseName` regex `^[a-zA-Z0-9_]+$` on 6 endpoints
8. **UI Offline window after daemon restart** — fast-retry cascade 300–1500ms
