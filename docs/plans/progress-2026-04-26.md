### Pattern lessons accumulated this batch

- **Eligibility predicates as named functions** — `canRollback(row)` named function over inline ternary; clarifies intent + auditable + unit-testable
- **Client-side filtering of bounded server lists** — when server caps results, filter client-side; faster UX, simpler code
- **Confirmed event payload extension** — use typed object `{snapshot: boolean}` rather than parent refs; modal stays self-contained
- **Host-aware defaults** — for asymmetric-risk options, default ON for `/prod/i` hosts, OFF otherwise
- **Daemon→client SSE payload enrichment for UI urgency** — embed source-of-truth timestamps in the SSE event rather than refetch
- **Auto-dismiss invalidated UI state** — tick-poll validity + dismiss-with-toast when state goes stale
- **JSDoc asterisk-emphasis breaks esbuild** — never use `*foo*` in TypeScript JSDoc bodies; the `/* */` terminator is whitespace-insensitive
